# 🏥 Hospital Emergency Room Dashboard

## 📌 Overview
This project is an interactive Hospital Emergency Room Dashboard developed using **Power BI**. It helps analyze patient flow, emergency room performance, wait times, admissions, referrals, and patient demographics through interactive visualizations.

## 📊 Dashboard Features
- Total Patients
- Average Wait Time
- Patient Satisfaction Score
- Patient Referrals
- Admission Status
- Patients Seen Within 30 Minutes
- Age Group Analysis
- Gender Analysis
- Race Analysis
- Department Referrals
- Patient Visits by Day
- Patient Visits by Hour

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
